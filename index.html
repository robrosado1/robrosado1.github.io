Pets vs Skeletons

Pets vs Skeletons is a tower defense style game using a grid- based placement system. The entirety of the game and its logic was created using JavaScript; in particular, a huge emphasis on class-based inheritance for all in-game entities and their interactions. All styling was created using HTML and CSS on JavaScript's Canvas feature using requestAnimationFrame to set the speed of rendering.
