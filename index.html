LikeMe

A pixel-perfect recreation of a popular one-page, social media website.

Using Ruby on Rails with PostgreSQL on the backend this app can support a multitude of users and all their relevant data with ease. With the use of custom SQL queries I can limit the number of unnecessary hits to the database. React and Redux on the frontend also let me authenticate users with bootstrapping. Modular components for reusability makes for more DRY-code and allows a more streamlined user experience.

  • Created: December 2018
  • Visit: LikeMe
  • Repo: Github

Pets vs Skeletons

My browser-based JavaScript game based on a popular mobile game.

Pets vs Skeletons is a tower defense style game using a grid-based placement system. The entirety of the game and it's logic was created using JavaScript; in particular, a huge emphasis on prototypal inheritance for all in-game entities and their interactions. All styling was created using HTML and CSS on JavaScript's Canvas feature using requestAnimationFrame to set the speed of rendering.

You can play Pets vs Skeletons here or check the repo here.

Radical

Inspired by the Rails framework, a more lightweight View-Controller with the same functionality.

Radical was made to be used and operate in the same way Rails does. Made with a lot of Object Oriented Programming as well as Metaprogramming. Designed to keep work with the request-response cycle easy and familiar.
